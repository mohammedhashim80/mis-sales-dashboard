# Company MIS Dashboard — Automated Sales Reporting

*Tools:* Excel · Power BI · DAX · Power Query  
*Region:* UAE & KSA

---

## Overview
Built a Monthly MIS Reporting System tracking 240+ sales transactions 
across UAE & KSA. Automated KPI reporting, regional analysis, and 
sales rep scorecards — saving 5+ hours per reporting cycle.

---

## Key Features
- Revenue vs Target tracking (12 months)
- Regional breakdown — Dubai, Abu Dhabi, Sharjah, Riyadh, Jeddah
- Sales rep performance scorecard
- Category analysis — 5 product categories
- Fully formula-driven — no macros needed
